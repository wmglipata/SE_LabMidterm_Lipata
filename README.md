## 📌 Project Overview
This repository contains a modular **User Authentication Validator** built in Python. The goal of this project is to demonstrate professional software engineering practices, including modular design, automated testing, and Continuous Integration (CI).

## 🛠️ Features
- **Email Validation:** Uses Regex to ensure proper email formatting.
- **Password Strength Check:** Validates minimum length and numerical requirements.
- **Error Handling:** Implements proper input validation and exception raising.

## 📁 Repository Structure
- `src/`: Contains the core application logic.
- `tests/`: Contains automated test cases (Positive, Negative, Edge).
- `.github/workflows/`: CI pipeline configuration.
- `quality_analysis.md`: ISO/IEC 25010 analysis.
